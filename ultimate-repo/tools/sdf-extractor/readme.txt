Утилита, генерящая sdf-ки по старым xml-кам редакторов (с svg-шной графикой). 
Умеет генерить sdf-ку по отдельной svg-шке.
Использование:
1. sdfextractor <имя xml-ки>
например sdfextractor usecase_metamodel.xml
должна появиться папка svgs с svg-шками, извлечёнными из xml-ки, 
и папка sdfs с соответствующими sdf-ками.
2. sdfextractor <имя sdf-ки>
по указанной svg-шке генерирует sdf-ку с тем же именем, кладёт в текущую папку.
